# Security Policy

## Supported Versions

We currently provide security updates for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of dotnet-grpc-gateway seriously. If you discover a security vulnerability, please let us know immediately.

Please report security vulnerabilities using one of the following methods:

1. **GitHub Private Advisory:** Use GitHub's built-in private advisory feature in the repository's Security tab to report the vulnerability privately.
2. **Email:** Contact us directly at rutova2@gmail.com.

We will acknowledge receipt of your vulnerability report within 48 hours and strive to send you regular updates about our progress. If you have not received a reply within 48 hours, please reach out again.
