# Docker Deployment Example for v2.0

This example demonstrates how to deploy the dotnet-grpc-gateway library in a Docker container.

## Prerequisites

* Docker installed on your machine
* .NET Core 3.1 or later
* gRPC-Web

## Usage

1. Create a new .NET Core project.
2. Install the dotnet-grpc-gateway NuGet package.
3. Configure the gateway.
4. Create a Dockerfile.
5. Build the Docker image.
6. Run the Docker container.
