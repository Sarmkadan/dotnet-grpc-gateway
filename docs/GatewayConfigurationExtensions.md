# GatewayConfigurationExtensions

`GatewayConfigurationExtensions` provides a set of static utility methods for querying and manipulating configuration settings related to gRPC gateway endpoints in the `dotnet-grpc-gateway` project. These extensions operate on instances of `GatewayConfiguration` or its derived types, offering conveniences for retrieving endpoint URLs, SSL settings, request limits, and monitoring configurations without exposing direct property access.
